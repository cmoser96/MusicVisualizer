# Music Visualizer

This is a music visualizer built using OpenGL, PulseAudio, and FFTW3.

## Installing dependencies

`$ sudo apt-get install libpulse-dev libfftw3-dev`

## Building

```
$ mkdir build && cd build
$ cmake ..
$ make
```

## Running
From within the */build* directory

`$ ./musicvisualizer.run`
